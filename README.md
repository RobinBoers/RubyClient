# RubyClient
Minecraft Bedrock Edition port of TheFREDProject. 

#### Features
- Clean UI (from Astral Client)
- Better sun and moon
- Red flowers on Iron Golem
- Plain leather armor
- Smoother Stones
- True White Snow
- Quiter everything
- Oreborders
- Transparent UI (optional)
- Clean Redstone Dust
- Borderless Glass
- Unobtrusive textures
- No Fog
- Less Pumkin Blur
- Better bedrock
- Cyan Slimes
